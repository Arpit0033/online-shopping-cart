# Online Shopping Cart – Python (OOP Based)

This is a console-based Online Shopping Cart application developed using
Python and Object-Oriented Programming (OOP) concepts.  
The project simulates basic shopping cart functionality such as adding
products, removing products, and calculating the total bill.

## Features
- Add products to the shopping cart
- Remove products from the cart
- View cart items
- Calculate total price
- Fully object-oriented design

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Code reusability
- Modular design (logical separation using classes)

## Technologies Used
- Python 3
